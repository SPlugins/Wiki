# Custom Items & Blocks for Minecraft Servers

Create **custom items**, custom blocks, custom events, and custom furniture for your Minecraft server — no coding required. Trusted by 4,000+ servers worldwide.

## 🎮 Custom Items & Blocks Plugins

- **[ExecutableItems](/executableitems/information-ei)** - The #1 custom items plugin for Minecraft. Create custom weapons, tools, armor, and items with unique abilities, custom textures, and crafting recipes
- **[ExecutableBlocks](/executableblocks/information-eb)** - Create custom blocks with special functions, animations, and interactions
- **[ExecutableEvents](/executableevents/information-ee)** - Create custom events and triggers for your server
- **[MyFurniture](/myfurniture/information-mf)** - Add custom furniture and decorations to your Minecraft server
- **[ExecutableCrafting](/executablecrafting/information-ec)** - Create custom crafting recipes for any item

## 🛠️ Tools for All Plugins

All plugins use SCore, a shared engine with powerful tools:

- **[Custom Commands](/tools-for-all-plugins-score/custom-commands)** - Execute complex command sequences
- **[Custom Conditions](/tools-for-all-plugins-score/custom-conditions)** - Create conditional logic for your custom items and blocks
- **[Custom Projectiles](/tools-for-all-plugins-score/custom-projectiles)** - Design unique projectile behaviors
- **[SCore Variables](/tools-for-all-plugins-score/score-variables)** - Dynamic variable system for storing player and global data
- **[Placeholders](/tools-for-all-plugins-score/placeholders)** - Dynamic placeholders for player data, entities, blocks, and more
